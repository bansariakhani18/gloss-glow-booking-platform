# Gloss & Glow Booking Platform

A full-stack booking and management platform built for **Gloss & Glow Car Detailing Studio**. The application allows customers to book detailing services online while providing administrators with a modern dashboard to manage appointments, services, and bookings.

---

## Features

### Customer Side

* Online appointment booking
* Service catalogue with pricing and descriptions
* Gallery showcase
* Responsive mobile-friendly interface
* Booking validation
* Automatic scroll restoration between pages

### Admin Side

* Secure admin authentication
* Dashboard with booking statistics
* Appointment management
* Add, edit, activate and deactivate services
* Search appointments
* Status management
* Delete appointments
* Real-time dashboard updates

### Backend

* REST API built with Flask
* SQLite database
* Session-based authentication
* Backend validation
* Slot capacity management
* CORS configuration
* Environment variable support

### Testing

* Automated booking validation test suite
* Backend input validation
* CRUD testing
* API testing
